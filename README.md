# Juego Arkanoid

Juego de Arkanoid en HTML, CSS y JavaScript con **cero dependencias**, jugable en el navegador.
Ya hay un **MVP jugable**: mueve el paddle, lanza la pelota, rompe todos los bloques y evita
perder tus 3 vidas.

## Cómo ejecutar

No hay paso de build. Sírvelo con cualquier servidor estático local desde la raíz del proyecto
y abre la página en el navegador. (Abrir `index.html` con `file://` directamente puede impedir
que carguen la imagen de sprites y los sonidos, por eso conviene un servidor estático.)

## Controles

- **← / →** — mover el paddle.
- **Espacio** — lanzar la pelota; y reiniciar desde las pantallas de victoria o game over.
